Chika Server～Conquer Depths, Live Depths～

Build a brand-new town in the underground world!

Chika Server is a survival server focused on underground building. Surface construction is strictly prohibited.

Server features

  • Surface building is prohibited: All construction must be done underground.
  • Take on the challenge of underground construction: Unleash your imagination below the surface.
  • Efficient resource use: No resets or separate resource worlds.
  • Open to global players: The server is primarily in Japanese, but English-speaking players are welcome and supported.
