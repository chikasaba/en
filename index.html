Chika Server～Conquer the Depths, Live in the Depths～

Build a brand-new town in the underground world!

Chika Server is a minecraft server focused on underground building. Surface construction is strictly prohibited.

Server Features

  • Strictly Underground: All construction must be kept below the surface. We preserve the natural terrain.
  • Subterranean Development: Put your imagination to the test with underground cities, bases, and secret bunkers.
  • Sustainable World: No resets or separate resource worlds. Mining and building all happen in one shared space.
  • Global Community: While primarily Japanese-speaking, we warmly welcome and support English communication.
